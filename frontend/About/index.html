About Labsim

Labsim was created by three developers from MRSM Langkawi. It is created to help users replicate experiment seamlessly. This simulator is created through multiple platforms to allow students replicate lab experiments accurately. Labsim is using IGCSE Biology 2026 syllabus as reference. Biology teachers can also use this platform in class as study aids to help students comprehend the topic more or if the teacher has no materials or apparatuses available.
